2023 OKR check

最近更新时间： 2023年4月23日。

O1：[80%]【事业发展】成功转到开发，有相关的工作经验， 养成持续学习的习惯。
KR1: 有一段开发或工作的经历。
[100%] 在快手商业化实习了不到三个月的时间, 虽然比较摸鱼, 但是这段实习经历还是对我有不小帮助(主要在团队开发和Java的库管理). 第二段在TUe的实习也刚刚开始, 内容是关于LLM自动修复JAVA API的breaking changes. 来TU/e之后对软件领域有了更深刻的理解, 我非常喜欢这里的课程设置, 了解到了software verification和evolution这些之前没有接触过的概念.
KR2: 基础知识牢固。力扣刷300题。Java面经和框架的知识掌握牢固，需要去扒底层代码。
[50%] Leetcode只刷到了一百多道题. Java 面经的熟练程度能有70%但是都是用中文来学习的, 到现在才发现还需要用英文再准备一遍.
KR3: 硕士成功入学，课程跟上。
[100%] TU/e成功入学, Q1没有挂科, Q2放弃了一门课, 两个Q应该能收获35学分, 目前均分在8分.

O2：[90%]【身心健康】保持心理健康，坚持运动降低体脂（以观赏性为标准）。
KR1: 留学之后打开自己， 尝试接触新的环境，明确课题划分的标准。
[85%] 我越来越勇敢和乐观. 课题划分的概念更加熟练, 能够经常运用到待人接物中来.
KR2: 拒绝内耗。找到一个统一的方法理论。
[80%] 对于情绪状态非常敏感, 能够很快的意识到自己的状态. 但是在情绪超down的情况下也很难保持理性.
KR3: 减脂。增加有氧运动，力量训练的频率可以低一些但必须保持。
[30%] 来荷兰之后课业压力太多觉得自己要寄掉了, 去健身房的次数也大大减少. 新的一年在保证自己精力的前提下多去跑步, 力量训练的频率可以适量较少.

O3：[20%]【技艺才情】练习游泳，精进视频剪辑技术，提升厨艺。
KR1: 练习游泳，每两周一次。
[20%] 来荷兰后并没有坚持下来.
KR2: 自己尝试做75道不同的菜，掌握几道拿手菜。
[60%] 没有统计数量, 但是也是有几个拿手菜的. 做饭也越来越简单越来越倾向于白人饭.
KR3: 系统地学习视频剪辑技术。
[0%] 下半年没有跟进OKR, 导致这个KR跟丢了.
KR4: 开通自己的播客频道， 并用它来保证自己每天的阅读量。
[5%]一个空有热情但并没有实行下去的KR.
